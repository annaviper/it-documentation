There are two categories of ports:
- Transmission Control Protocol (TCP)
- User Datagram Protocol (UDP).

| Port(s)       	| Protocol        	|
|---------------	|-----------------	|
| 20/21 (TCP)   	| FTP             	|
| 22 (TCP)      	| SSH             	|
| 23 (TCP)      	| Telnet          	|
| 25 (TCP)      	| SMTP            	|
| 80 (TCP)      	| HTTP            	|
| 88             	| Kerberos         	|
| 161 (TCP/UDP) 	| SNMP            	|
| 389 (TCP/UDP) 	| LDAP            	|
| 443 (TCP)     	| SSL/TLS (HTTPS) 	|
| 445 (TCP)     	| SMB             	|
| 3389 (TCP)    	| RDP             	|
